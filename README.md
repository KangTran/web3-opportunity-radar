# 🔍 Web3 Opportunity Radar
Track early-stage Web3 projects, testnets, and potential airdrops before they trend ⚡

## 🌐 Live Demo
👉 **[Open Web3 Opportunity Radar](https://kangtran.github.io/web3-opportunity-radar/)**

A lightweight tool to track early-stage Web3 opportunities including testnets, airdrops, and new protocol launches.

## 🚀 Overview

Web3 Opportunity Radar helps early users and researchers identify asymmetric opportunities before they trend.

Built as a minimal prototype demonstrating:

- Opportunity aggregation
- Lightweight frontend dashboard
- Structured data tracking
- Builder mindset for early discovery tools

## 🧠 Features

- 📡 Track new protocols and ecosystems
- 💰 Monitor potential airdrops
- ⚡ Lightweight and fast
- 🌐 Runs entirely in the browser
- 🧩 Easy to extend

## 🛠️ Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- JSON data storage

Designed for:
